# 🎯 SEO Analysis Venue Dashboard

A **Streamlit-based SEO intelligence dashboard** that analyzes website performance, benchmarks competitors using Google search data, identifies keyword opportunities, and generates AI-powered SEO insights.

---

## 🚀 Features

### 🔍 SEO Analysis
- Title, Meta Description, H1 extraction
- Keyword density and word count
- Internal & external link analysis
- Image ALT tag detection

### ⚡ Performance Insights
- Google PageSpeed integration
- Technical SEO scoring

### 🥇 Competitor Analysis
- Google SERP competitor discovery
- Direct competitor filtering
- Ranking-based comparison

### 📊 Benchmarking
- SEO score comparison across competitors
- Position tracking
- Gap identification

### 🔑 Keyword Opportunities
- Finds missing keywords vs competitors
- Detects high-frequency competitor terms
- Works across industries (universal SEO)

### 🤖 AI Executive Summary
- Generates business-level SEO insights
- Highlights ranking gaps and strategies
- Provides actionable recommendations

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Requests
- BeautifulSoup
- Google APIs (PageSpeed, Gemini)
- Serper API (Google Search)
